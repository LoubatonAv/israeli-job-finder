# AI Upgrade Summary

מה עודכן בגרסה הזו:

- עברית מלאה יותר ב־UI ובפלטים המרכזיים.
- תיקון סינון רעש מ־JobMaster / AllJobs: דפי קטגוריה, עמודים שלמים וכפתורי UI כבר לא אמורים להיכנס כמשרות.
- שיפור חילוץ מיקום: לא משתמשים יותר ב־sourceQuery כדי לא להדביק בטעות "חיפה" לכל משרה.
- שיפור חילוץ חברה מ־Drushim / JobMaster / AllJobs.
- שיפור חילוץ סוג תפקיד: לא כל "בודק/ת" נחשב QA; צריך הקשר של תוכנה / QA / מערכת / tester.
- סינון תפקידים לא רלוונטיים: ייצור, אח/ות, חשבונות, מכונאות, שרטוט, תשתיות, מזון וכו׳.
- Scoring חדש בעברית שמעדיף QA Junior, בדיקות ידניות, מערכות מידע, מיקומי צפון/חיפה/קריות/יקנעם, ומוריד ניקוד למשרות בכירות/טלפוניות/משמרות/מכירות.
- Feedback learning בטוח יותר: שומר locationKey/roleType ולא לומד מ־Israel כללי.
- API client עמיד יותר למצב שבו השרת מתאפס באמצע בקשה ולא מחזיר JSON.
- הוסר קובץ server/.env כדי לא לכלול סודות או API keys בתוך ZIP.

בדיקות שבוצעו:

- node --check לקבצי שרת מרכזיים.
- npm --prefix client run build עבר בהצלחה.
- בדיקת scoring סינתטית לכמה דוגמאות.

המלצה להפעלה ראשונה:

1. העתק `server/.env.example` אל `server/.env`.
2. התחל עם `SEARCH_PROVIDERS=drushim,matrix`.
3. אחרי שהכול נראה טוב, אפשר לבדוק JobMaster בנפרד עם DEBUG_DRY_RUN=true.
